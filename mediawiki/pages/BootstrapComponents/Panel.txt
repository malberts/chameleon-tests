== Description ==
A panel in bootstrap is a bordered box with some padding around its content.

For available options to this component, please see the [https://github.com/oetterer/BootstrapComponents/blob/master/docs/components.md#panel documentation on github].

=== Color variants ===
Panels support the following coler codes:
* default
* primary
* success
* info
* warning
* danger
